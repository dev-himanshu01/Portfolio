Project Overview

This is written with the idea to provide management platform for Human Resource Managers and, the admins of the company. This application works out the concepts of a full stack web Application without short-comings in the manual file system which was called sanchez application.

The Company Information Tracking System (CITS) is a comprehensive web-based application developed using Java, HTML, CSS, JavaScript, Apache Tomcat Server 8.2, and Oracle Database. CITS aims to streamline and enhance the management of company-related information, catering to the needs of administrators, customers, and managers. The system offers varying levels of access and functionality based on user roles, ensuring efficient information tracking, analysis, and reporting.
